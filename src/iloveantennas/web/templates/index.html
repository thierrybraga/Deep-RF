Ajuda - IloveAntenas

Tipos de Antena

  • Dipolo: Antena básica λ/2, omnidirecional no plano H. Ideal para radiodifusão e recepção geral.
  • Monopolo: Metade do dipolo sobre plano de terra (λ/4). Comum em veículos e dispositivos portáteis.
  • Yagi-Uda: Array direcional com ganho. Usada para TV e comunicações ponto-a-ponto de longa distância.
  • Patch: Antena planar para aplicações compactas (GPS, WiFi, Celulares).
  • Helicoidal: Polarização circular, alta diretividade. Usada em comunicação via satélite.

Fórmulas Principais

  • Comprimento de Onda: λ = c/f (onde c ≈ 3x10⁸ m/s)
  • Impedância (Z): Z = R + jX (Resistência + Reatância)
  • Return Loss (S11): Medida em dB da potência refletida. Menor é melhor (ex: -20dB).
  • VSWR: Razão de onda estacionária. Ideal é 1.0. Aceitável < 2.0.
